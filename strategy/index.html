load_provisioning()

Method to load data on object initialization, if desired. This should parse the raw values in self.parameters into format Python can comprehend.

The value returned by load_provisioning() will be stored in the self.parsed_provisioning class variable when object is created. The superclass returns an empty list since most of Unleash's default strategies are list-based (in one way or another).
